Projects I've Done

TMM

Developed logistics operations application that provide an integrated set of facility terminal management tools designed to assist users with the execution requirements and business processes for the management of transportation terminals.

Using multi-agent architecture, applying expert load planning knowledge, and creating effective load plans for yard, air, rail, and ship operations.

Leveraging Java with Spring MVC back-end, PostgreSQL with Hibernate Model database, Flash front-end.
